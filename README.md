# Personal Task Management App

This is a Task Management Application built using React and Vite.

## 🚀 Features
- Create Task
- Edit Task
- Delete Task
- Task Status (Pending, In Progress, Completed)
- Responsive Card UI
- Data persistence using localStorage
- Bug fix using functional state update
- Dynamic date using API

## 🛠️ Tech Stack
- React (Vite)
- JavaScript
- CSS

## 📦 How to Run

```bash
npm install
npm run dev
